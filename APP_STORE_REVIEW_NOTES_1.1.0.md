How to test (subscriptions):
Log in with the test account.

Go to Profile -> Subscription (or during onboarding: "Choose Subscription").

Select any plan and complete the purchase using a Sandbox Apple ID.

After purchase, subscription features should unlock automatically without restarting the app.

If needed, tap "Restore Purchases" on the subscription screen.

Available subscriptions:
Player Basic
Player Premium
Trainer Basic
Trainer Standard
Trainer Premium

Subscriptions can be managed/cancelled in the user's Apple ID settings.
No external payments are used; all purchases are handled via Apple In-App Purchase.
